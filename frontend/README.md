# Customer Service AI Frontend

Next.js frontend for the Customer Service AI Agent application.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

Make sure the backend API is running on `http://localhost:8000` or update the API URL in `app/page.tsx`.
